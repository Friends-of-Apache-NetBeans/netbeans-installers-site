Installers and packages for Apache NetBeans

Apache NetBeans packages / installers provided by Friends of Apache NetBeans (see who we are). These packages are built with NBPackage using the official Apache NetBeans binary zip release. The packages themselves are not released by the Apache Software Foundation.

Most installers include a local Java JDK for the IDE to run on, offering a self-contained out-of-the-box experience. Additional JDK's for projects may be registered and/or downloaded using Tools / Java Platforms in the IDE menu, supporting development of projects from Java 8 up.

For each supported Apache NetBeans release there is also a JDK-less version per platform. This requires that your machine already has a JDK installed.

To get the proper package or installer for your platform, select one of the tabs above or one of the links below.

deb_arm64
Debian for ARM64 machines such Raspberry Pi with Raspberry-OS.
deb_x64
Debian for Intel and AMD PCs and laptops.
macOS_ARM64
Apple macOS of ARM, a.k.a. Apple Silicon (M1, M2, M3, M4, M5,..).
macOS_x64
Apple macOS on Intel 64 bit
rpm_x64
Red Hat Package Manager (RPM) for Intel/AMD 64 bit
Windows_x64
Microsoft Windows for Intel/AMD 64 bit

Support

For issues with the installers / packages themselves, please raise an issue on our GitHub repository.

For community support with the IDE itself, please use the Apache NetBeans mailing lists.
